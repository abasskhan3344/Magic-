## 🚀 Magic is live

Magic is the AI agent for vision-based UI automation across web, mobile, desktop, and hybrid applications. Write natural-language workflows, automate UI actions, and validate what users actually see.

## ✨ Why Magic

Modern UI automation is fragile when it relies on selectors, DOM structure, or accessibility trees. Magic works from screenshots and visible interface state so it can:

- **Reduce maintenance** — no fragile selectors to update after UI changes.
- **Reach every visible surface** — if a human can see it, Magic can act on it.
- **Validate real UI experience** — assert visual state, layout, and rendered output.
- **Work across platforms** — web, Android, iOS, desktop, and custom interfaces.

## 💡 What you can automate

Magic works anywhere you can capture the screen. Use it to automate browsers, apps, native interfaces, or any UI surface that is visible on a display.
