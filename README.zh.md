## 🚀 Magic 已上线

Magic 专注于可视化 UI 自动化，可应用于网页、移动、桌面和混合应用。你可以使用自然语言描述任务，自动执行界面操作并验证视觉结果。

## ✨ 为什么选择 Magic

依赖选择器、DOM 或辅助功能树的自动化往往很脆弱。Magic 直接从屏幕截图和可见界面状态工作，因此它可以：

- **降低维护成本** — 当 UI 更改时，不再依赖易碎选择器。
- **触达所有可见界面** — 只要人眼能看到，Magic 就能操作。
- **验证真实界面体验** — 校验视觉状态、布局和渲染结果。
- **跨平台运行** — 支持网页、Android、iOS、桌面和自定义界面。
